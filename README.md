# Python Logging Choices Comparison

This example demonstrates how to make smart "free choices" in open-source tools by comparing two common logging approaches in Python: simple `print()` statements and the more robust standard `logging` module. It highlights the trade-offs in terms of simplicity, configurability, and suitability for different project scales, even when both options are "free" (built-in).

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
